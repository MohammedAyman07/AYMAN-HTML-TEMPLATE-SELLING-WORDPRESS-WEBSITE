GETTING STARTED
--------------------------------------------------------------------------------

1. Extract all files to your web server or local host directory
2. Open index.html in a modern browser (Chrome, Firefox, Edge, Safari)
3. Replace placeholder images in the /images/ folder with your own artwork
4. Edit text content directly in each HTML file
5. Customize colors and fonts via CSS variables in css/style.css (:root section)


--------------------------------------------------------------------------------
CUSTOMIZATION GUIDE
--------------------------------------------------------------------------------

COLOR SCHEME
  Open css/style.css and edit the :root variables at the top:
  
  --primary:       #7c3aed;   тЖР Main purple accent (change to any color)
  --secondary:     #f97316;   тЖР Orange accent
  --dark:          #0f0f0f;   тЖР Dark background
  
FONTS
  The template uses Google Fonts:
  - Inter (body text)
  - Playfair Display (headings)
  
  To change fonts, update the Google Fonts link in each HTML <head> section
  and change --font-main and --font-display in the CSS :root.
  
GALLERY IMAGES
  Replace the src attributes of <img> tags inside .gallery-item elements.
  Update data-src and data-caption attributes accordingly.
  Also update the data-category attribute for filter functionality:
  - digital
  - illustration
  - painting
  - concept

CONTACT EMAIL
  Replace "aymaninfotechs@gmail.com" with your own email address across all pages.
  To make the contact form actually send emails, integrate a backend service
  such as Formspree, EmailJS, or your own PHP mailer.
  
SOCIAL LINKS
  Update the href="#" links in the footer and contact sections
  with your actual social media profile URLs.
