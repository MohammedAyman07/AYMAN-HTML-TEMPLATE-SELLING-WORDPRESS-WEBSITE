Creative Designer Portfolio - HTML Template
===========================================

Overview
--------
This is a modern, creative, and fully responsive portfolio website template built with HTML5, CSS3, Bootstrap 5, and JavaScript. The template is tailored for graphic designers, UI/UX designers, illustrators, and creative professionals.

Folder Structure
----------------
creative-designer-portfolio/
├── css/             - Contains customized style.css
├── js/              - Contains custom script.js
├── images/          - Contains placeholder images (replace with yours)
├── index.html       - Main Landing Page
├── about.html       - Designer details and skills
├── portfolio.html   - Project Showcase Grid
├── services.html    - Services and Testimonials
├── contact.html     - Contact Form and info
└── readme.txt       - Documentation file

Features
--------
- Creative & artistic layout with colorful UI
- Unique shapes and smooth hover animations
- Fully responsive across devices
- Clean, SEO-friendly HTML structure
- Sticky navigation and custom reveal-on-scroll effects
- Based on Bootstrap 5

Customization
-------------
Styles:
All primary colors, gradients, and font families are stored as CSS variables at the top of the `css/style.css` file. Simply change the values to update the template's look.
Example:
:root {
  --primary: #8a2be2;
  --secondary: #ff1493;
  --tertiary: #00d2ff;
}

JavaScript:
All scripts for smooth scrolling, sticky navigation, and scrolling reveal animations are located in `js/script.js`.

Contact Form:
By default, the contact form includes a mock submission state in `script.js`. You will need to connect this to a backend script (e.g., PHP, Formspree, etc.) to process emails to aymaninfotechs@gmail.com.
