================================================================================
  TEMPLATE OVERVIEW
================================================================================

This is a clean, modern, and fully responsive freelancer portfolio website
template built with HTML5, CSS3, Bootstrap 5, and vanilla JavaScript.
Designed for freelancers, remote professionals, consultants, and independent
service providers who want to showcase their services and projects online.

================================================================================
  CUSTOMIZATION GUIDE
================================================================================

1. BRAND NAME
   Search for "Freelance" / "Freelancer" in all HTML files and replace with
   your own name or brand name. Also update <title> tags and meta descriptions.

2. PROFILE IMAGE (HERO)
   Replace the src URL in index.html hero section:
     <img src="https://images.unsplash.com/..." alt="..." />
   Place your photo in the /images/ folder and update the src path:
     <img src="images/your-photo.jpg" alt="Your Name" />

3. CONTACT EMAIL
   Search for "aymaninfotechs@gmail.com" across all files and replace with
   your own email address.

4. SOCIAL MEDIA LINKS
   Update all <a href="#"> social links throughout the HTML files with your
   actual profile URLs.

5. PORTFOLIO IMAGES
   Replace Unsplash URLs in portfolio cards with your own project screenshots:
     <img src="images/project-name.jpg" alt="Project Name" />

6. TESTIMONIAL PHOTOS
   Replace Unsplash avatar URLs in testimonial cards with real client photos:
     <img src="images/client-name.jpg" alt="Client Name" />

7. SKILLS & PERCENTAGES
   In skills.html / index.html, adjust the data-width="XX%" values on each
   .skill-bar-fill element to match your actual proficiency levels.

8. PRICING
   Update pricing values and feature lists in services.html to match your
   actual service packages and rates.

9. COLORS
   Edit the :root { } CSS variables in css/style.css to change the theme.
