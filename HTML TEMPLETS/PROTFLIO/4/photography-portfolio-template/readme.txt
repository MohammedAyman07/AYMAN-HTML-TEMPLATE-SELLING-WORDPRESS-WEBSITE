# Lens & Light - Photography Portfolio Template

## Description
A modern, minimal, dark-themed photographer portfolio website template. Built with HTML5, CSS3, Bootstrap 5, and JavaScript. Ideal for photographers, videographers, visual storytellers, and creative studios.

## Features
- Minimalist and Elegant Design
- Fully Responsive (Mobile-first approach)
- Dark Background Theme with Gold Accents
- Bootstrap 5 Framework
- GLightbox Multi-media Gallery
- Google Fonts (Outfit & Playfair Display)
- Clean and well-commented code
- W3C Valid HTML

## File Structure
- `index.html`: Home page with hero section, latest work, about snippet, services snippet, testimonials.
- `gallery.html`: Portfolio page with filterable masonry-like grid.
- `about.html`: Photographer introduction, timeline, and artistic style.
- `services.html`: Detailed service offerings and what is included.
- `contact.html`: Contact details and functional-looking enquiry form.
- `css/style.css`: All custom styling for colors, layouts, and animations.
- `js/script.js`: Initialization of lightbox, filtering mechanisms, and sticky nav listener.

## Usage Instructions
1. Open any of the HTML files in your browser.
2. Customize the content (heading texts, descriptions, email to `aymaninfotechs@gmail.com`).
3. Replace the Unsplash image sources with your own optimized photography for improved load times. Note: Currently, images are loaded statically via high-quality Unsplash URLs to instantly demonstrate template beauty out-of-the-box.
