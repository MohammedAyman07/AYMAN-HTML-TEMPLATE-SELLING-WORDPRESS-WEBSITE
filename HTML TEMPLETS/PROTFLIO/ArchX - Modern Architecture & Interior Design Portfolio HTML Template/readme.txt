Architect Portfolio Template
============================

Description:
A modern, elegant, and minimal HTML5 portfolio template designed for architects, interior designers, and construction studios.

Features:
- Fully Responsive Design (Bootstrap 5)
- Clean & Minimal Architecture Aesthetic
- Project Portfolio Grid
- Smooth Hover Animations
- Sticky Navigation
- Google Fonts (Playfair Display, Montserrat)
- SEO Friendly

Folder Structure:
- index.html (Main Entry)
- about.html
- projects.html
- services.html
- contact.html
- css/
  - style.css (Custom styling)
- js/
  - script.js (Interactivity)
- images/ (Placeholder for images)

Usage:
1. Open index.html in any modern web browser to view the template.
2. Edit HTML texts and links as needed.
3. Replace image URLs with your own project images. Check the "images" folder.
