DESCRIPTION
-----------
A modern, dark-mode developer portfolio website template built
with HTML5, CSS3, Bootstrap 5, and Vanilla JavaScript. Designed
for software developers, programmers, engineers, and tech freelancers.

FOLDER STRUCTURE
----------------
developer-portfolio-template/
│
├── index.html         — Main page (Hero, Skills, Projects,
│                         Experience, Testimonials, Contact)
├── projects.html      — Full projects grid with filter
├── skills.html        — Detailed skills with progress bars
├── experience.html    — Career timeline & education
├── contact.html       — Contact form with FAQ accordion
│
├── css/
│   └── style.css      — Global stylesheet (dark theme,
│                         neon accents, animations)
├── js/
│   └── script.js      — All interactivity (typed effect,
│                         scroll reveal, particles, filter,
│                         form handler, back-to-top, counters)
│
├── images/            — Place your own images here
│
└── readme.txt         — This file

HOW TO USE
----------
1. Open any .html file in your browser — no build step needed.
2. Replace images with your own by updating <img src="..."> tags
   or placing files in the images/ folder.
3. Update personal info (name, email, bio, projects, experience)
   in each HTML file.
4. To connect the contact form to a backend/service (e.g. Formspree,
   EmailJS), edit the form submit handler in js/script.js.

CUSTOMIZATION
-------------
All design tokens (colors, fonts, spacing) are CSS custom properties
at the top of style.css inside the :root block. Change them once
to restyle the entire template:

  --accent-blue  : #00b4ff   /* Primary neon accent */
  --accent-green : #00ffb3   /* Secondary neon accent */
  --bg-primary   : #080f1a   /* Main page background */
