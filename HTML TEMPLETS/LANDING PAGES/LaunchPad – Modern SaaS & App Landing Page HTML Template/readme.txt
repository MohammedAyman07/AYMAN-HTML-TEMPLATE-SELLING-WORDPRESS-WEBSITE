GETTING STARTED
---------------
1. Open the "landing-page-template" folder.
2. Open "index.html" using any code editor.

Recommended editors:
  - Visual Studio Code   (https://code.visualstudio.com)
  - Sublime Text         (https://www.sublimetext.com)
  - Notepad++            (https://notepad-plus-plus.org)

Double-click "index.html" to preview it in your browser.


HOW TO EDIT TEXT
----------------
Open index.html and find the text you want to change inside HTML tags.

Example:
  <h1 class="hero-headline">High-Conversion Landing Pages</h1>

Change it to:
  <h1 class="hero-headline">Your Product Name Here</h1>

Repeat this for any other text sections (features, pricing, testimonials, etc.)


HOW TO CHANGE IMAGES
--------------------
1. Open the "images/" folder.
2. Replace existing images with your own using the same filename,
   OR place new images in the folder and update the src="" attribute in index.html.

Recommended image formats: .png, .jpg, .webp, .svg


HOW TO CHANGE COLORS
--------------------
Open css/style.css and look for the CSS variables at the top of the file:

  :root {
    --primary: #6366f1;        /* Main brand color */
    --primary-dark: #4f46e5;   /* Darker shade */
    --secondary: #06b6d4;      /* Accent/gradient color */
    ...
  }

Change the hex color codes (e.g. #6366f1) to any color you prefer.
Color picker tools: https://coolors.co  |  https://colorhunt.co


HOW TO CHANGE FONTS
-------------------
1. Visit https://fonts.google.com
2. Choose a font and copy the <link> tag.
3. Replace the existing Google Fonts link in the <head> of index.html.
4. In css/style.css, update --font-main and --font-body variables.


HOW TO EDIT SECTIONS
--------------------
Each section has a clear HTML comment label, for example:
  <!-- HERO SECTION -->
  <!-- FEATURES SECTION -->
  <!-- PRICING SECTION -->
  etc.

Find the section you want to edit and make changes within that block.


HOW TO ADD/REMOVE PRICING PLANS
--------------------------------
Each pricing plan is inside a <div class="col-md-6 col-lg-4"> block.
Copy, modify, or delete these blocks as needed.


HOW TO UPDATE CONTACT EMAIL
---------------------------
Search for "aymaninfotechs@gmail.com" in index.html and replace
all instances with your own email address.


HOW TO UPLOAD YOUR WEBSITE
--------------------------
1. Purchase web hosting (e.g. Bluehost, Hostinger, SiteGround)
2. Login to your hosting control panel
3. Open File Manager
4. Navigate to the "public_html" folder
5. Upload the entire contents of "landing-page-template/" folder
6. Your website will be live at your domain!

Recommended hosts:
  - Hostinger  (https://www.hostinger.com)
  - Bluehost   (https://www.bluehost.com)
  - Netlify    (https://www.netlify.com)  [Free hosting]
  - Vercel     (https://www.vercel.com)   [Free hosting]
