 HOW TO EDIT THE WEBSITE
------------------------------------------------------

1. Open the template folder.
2. Open index.html (or any page) in a code editor.

Recommended editors:
  - Visual Studio Code (https://code.visualstudio.com) [RECOMMENDED]
  - Sublime Text
  - Notepad++

Edit any text inside HTML tags to customize content.

------------------------------------------------------
 HOW TO CHANGE COLORS
------------------------------------------------------

Open css/style.css and look for the :root block near the
top of the file. You will find all color variables:

  --primary:       #5B5BFD   (main blue-purple)
  --secondary:     #8B5CF6   (purple accent)
  --accent:        #06B6D4   (cyan accent)
  --gradient-main: linear-gradient(135deg, #5B5BFD, #8B5CF6)

Change these values to update the color theme sitewide.

------------------------------------------------------
 HOW TO CHANGE IMAGES
------------------------------------------------------

Replace files inside the /images/ folder with your own.
Keep the same filenames, or update the src="..." attributes
inside the HTML files to match your new filenames.

------------------------------------------------------
 HOW TO CHANGE CONTACT EMAIL
------------------------------------------------------

Search for "aymaninfotechs@gmail.com" across all HTML files
and replace with your own email address.

 HOW TO DEPLOY YOUR WEBSITE
------------------------------------------------------

Option A – Traditional Web Hosting:
  1. Purchase a hosting plan (e.g., Hostinger, Namecheap)
  2. Log in to your hosting control panel (cPanel)
  3. Open File Manager
  4. Navigate to public_html/
  5. Upload all files from this folder
  6. Your site will be live at your domain

Option B – Netlify (Free & Fast):
  1. Go to https://netlify.com
  2. Drag and drop this entire folder into the deploy area
  3. Your site is live instantly with a free URL

Option C – GitHub Pages (Free):
  1. Create a GitHub account and new repository
  2. Upload all files to the repository
  3. Go to Settings → Pages → Deploy from main branch
  4. Your site will be live at username.github.io/repo-name
