Consulting Template Documentation

Overview:
This is a modern consulting business website template designed using HTML5, CSS3, Bootstrap 5, and JavaScript. The template is tailored for business consultants, financial advisors, marketing consultants, and professional service firms.

Features:
- Valid HTML5 & CSS3
- Fully Responsive Layout
- Clean, minimal, and professional design
- Sticky navigation bar
- Animated counters and smooth scrolling
- Custom CSS styling and typography (Inter & Montserrat)
- SEO Friendly

Folder Structure:
- index.html: The highly-converting landing and home page
- services.html: Dedicated services page
- case-studies.html: Page displaying successful projects
- team.html: Team directory
- faq.html: Frequently asked questions using Bootstrap accordion
- contact.html: Contact page with form and Google Maps embed
- css/style.css: Core stylesheet
- js/script.js: Core interactions and JavaScript file
- images/: Folder intended for all local assets

How to Use:
1. Extract the zipped folder.
2. Edit HTML files to customize text content.
3. Update images by placing new images in the "images/" folder and replacing the Unsplash URL links in the HTML code with local links relative to the "images" directory.
4. Customize `css/style.css` colors by modifying the :root variables at the start of the file.
