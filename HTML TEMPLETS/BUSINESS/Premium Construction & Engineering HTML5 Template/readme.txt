Template Name: ConstructionPro - Modern Construction Business Template
Version: 1.0
Author: AymanInfotech

Description:
ConstructionPro is a premium, modern, and fully responsive HTML5 template designed for construction companies, engineering firms, architecture studios, and industrial businesses. It features a bold industrial design with a strong typography and modern grid layout.

Features:
- HTML5 & CSS3
- Bootstrap 5 Framework
- Fully Responsive Design
- Modern and Strong Industrial Aesthetics
- Smooth Interactivity and Hover Animations
- SEO Friendly Structure
- Clean and Well Commented Code
- Animated Counter Statistics

File Structure:
- index.html (Home Page)
- about.html (About Us)
- services.html (Services Page)
- projects.html (Projects Gallery)
- team.html (Team Page)
- contact.html (Contact Form & Map)
- css/style.css (Custom Stylesheet)
- js/script.js (Interactive Scripts)
- images/ (Directory for local images)

Usage Instructions:
1. Extract the template files.
2. Open index.html in your web browser.
3. Replace the placeholder URLs in the <img> tags and css background images with your own local images from the /images folder.
4. Modify text and links as needed in the HTML files.
5. Setup the contact form backend script (PHP/Node.js) to receive emails (currently simulated with JavaScript alert).
