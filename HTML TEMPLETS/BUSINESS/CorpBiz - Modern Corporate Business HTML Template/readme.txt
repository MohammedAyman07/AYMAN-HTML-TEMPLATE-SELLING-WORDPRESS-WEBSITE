-----------------------------------------------------------------------------------
How to Use
-----------------------------------------------------------------------------------
1. Extract the zip file (if applicable) or copy the folder into your project directory.
2. Open `index.html` in your web browser to view the template.
3. You can edit the HTML structure directly in the `.html` files.
4. Custom styles can be modified in `css/style.css`. Root CSS variables are used for easy color theme changes.
5. All custom JavaScript functionalities (sticky navbar, counters, smooth scrolling) are in `js/script.js`.
