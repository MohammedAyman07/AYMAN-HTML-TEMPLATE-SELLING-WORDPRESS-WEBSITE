Setup Instructions for Startup Business Template
===============================================

1. Description
--------------
A modern, minimal, and conversion-focused HTML5 template designed for startups, tech companies, and digital agencies.

2. Features
-----------
- HTML5, CSS3, Bootstrap 5.3
- Fully Responsive Design
- Clean and beautifully commented Code
- Smooth CSS3 Animations & Scroll Animations (AOS)
- Isotope Portfolio Filtering
- Google Fonts (Inter)
- FontAwesome Icons

3. Folder Structure
-------------------
startup-business-template/
├── index.html        (Main landing page)
├── services.html     (Services list)
├── portfolio.html    (Portfolio / Projects)
├── pricing.html      (Pricing table)
├── about.html        (About us)
├── contact.html      (Contact form & Info)
├── css/
│   └── style.css     (Main custom stylesheet)
├── js/
│   └── script.js     (Custom interactivity & plugins initialization)
├── images/           (Placeholders & assets)
└── readme.txt
