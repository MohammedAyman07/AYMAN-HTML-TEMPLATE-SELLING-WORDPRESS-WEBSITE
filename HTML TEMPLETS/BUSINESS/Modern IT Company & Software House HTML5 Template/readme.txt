Modern IT Company - HTML5 Template
========================================

Theme Description:
-----------------
A premium, modern IT company business website template designed for IT service providers, software development firms, startups, and consultancies.

Main Features:
-------------
- Built with HTML5, CSS3, Bootstrap 5, and JavaScript
- Fully Responsive Design (Mobile-first)
- Modern and Professional UI
- Sticky Navigation Bar
- Animated Content (Counters)
- Clean and Well-Commented Code
- Google Fonts & FontAwesome Icons
