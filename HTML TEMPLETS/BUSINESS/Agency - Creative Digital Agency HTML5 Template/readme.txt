Digital Agency - Modern Business Template
=========================================

Description:
A modern, creative, and fully responsive HTML5 template designed for digital agencies, marketing agencies, creative studios, and web development companies. It features a dark-themed UI with vibrant gradient accents, smooth scroll animations.

Features:
- Built with HTML5, CSS3, Bootstrap 5.3, and Vanilla JS
- Fully Responsive Design (Mobile, Tablet, Desktop)
- Smooth Scroll Animations (AOS Library)
- Gradient Backgrounds & Vibrant Typography
- Sticky Navigation Navbar
- Functional Contact Form (JS validation placeholder included)
- Clean, Commented, and SEO-friendly Core Code

Folder Structure:
- index.html
- services.html
- portfolio.html
- pricing.html
- about.html
- contact.html
- css/
  - style.css     (Main stylesheet with root variables & custom CSS)
- js/
  - script.js     (Custom interactions and setup)
- images/         (Assets utilized across pages)

Installation & Usage:
1. Extract the downloaded ZIP folder.
2. Open `index.html` in your favorite web browser to preview it locally.
3. Edit the HTML files in any code editor (VS Code, Sublime, etc.) to replace text content.
4. Modify color variables in `css/style.css` at `:root` to match your brand.
