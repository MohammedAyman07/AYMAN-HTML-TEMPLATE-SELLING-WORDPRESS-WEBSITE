Place your course images, instructor photos, banner images, and other media files in this folder.

Recommended image sizes:
- Hero image: 900x600px
- Course card thumbnails: 600x400px
- Instructor photos: 400x400px (square)
- Blog post thumbnails: 800x500px

Supported formats: JPG, PNG, WebP, SVG

Note: The template uses Unsplash images via CDN in the demo.
Replace src attributes in HTML files with your own image paths.
