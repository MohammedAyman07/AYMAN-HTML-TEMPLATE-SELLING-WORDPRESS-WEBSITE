HOW TO EDIT THE WEBSITE
==================================================

1. Open the template folder.
2. Open index.html using a code editor.

Recommended editors:
- Visual Studio Code
- Sublime Text
- Notepad++

Edit text inside HTML tags to customize content.

==================================================
HOW TO CHANGE IMAGES
==================================================

Replace images inside the images/ folder with your own.
Make sure to keep the same filename or update the src
attribute in the HTML files accordingly.

Recommended image sizes:
- Hero image:     900x600px
- Course cards:   600x400px
- Instructor photos: 400x400px
- Blog thumbnails:  800x500px

==================================================
HOW TO CHANGE COLORS
==================================================

Open css/style.css and look for the :root section
at the top. Change the color codes to match your
brand colors.

Primary colors to change:
--primary:       #2563EB  (Main blue)
--accent:        #F59E0B  (Gold accent)

==================================================
HOW TO CHANGE FONTS
==================================================

The template uses Google Fonts: Poppins and Inter.
To change fonts, update the Google Fonts link in the
<head> section of each HTML file, and update the
font-family values in style.css.

==================================================
HOW TO UPLOAD WEBSITE
==================================================

1. Buy a web hosting plan
2. Login to your hosting control panel (cPanel)
3. Open File Manager
4. Navigate to public_html folder
5. Upload all template files and folders
6. Your website will be live!
