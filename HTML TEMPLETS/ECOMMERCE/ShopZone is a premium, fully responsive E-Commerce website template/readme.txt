------------------------------------------------------------
  HOW TO GET STARTED
------------------------------------------------------------

1. Open the "ecommerce-template" folder.
2. Double-click "index.html" to preview in your browser.

Recommended Code Editors:
  - Visual Studio Code (free) → https://code.visualstudio.com
  - Sublime Text              → https://www.sublimetext.com
  - Notepad++                 → https://notepad-plus-plus.org


------------------------------------------------------------
  HOW TO EDIT THE WEBSITE
------------------------------------------------------------

TEXT CONTENT:
  Open any .html file and find the text you want to change.
  Edit the text between HTML tags.
  Example: <h1>Your New Heading</h1>

CONTACT EMAIL:
  Search for "aymaninfotechs@gmail.com" across all pages
  and replace it with your own email address.

BRAND NAME:
  Search for "ShopZone" and replace with your store name.


------------------------------------------------------------
  HOW TO CHANGE IMAGES
------------------------------------------------------------

1. Prepare your images (JPG, PNG, or WebP recommended).
2. Rename them to match the existing file names:
     product-1.png, product-2.png, ... product-6.png
     hero-banner.png
3. Replace the old images inside the /images/ folder.

TIP: For best results, keep product images square
     (e.g., 600x600px) with a white or light background.


------------------------------------------------------------
  HOW TO CHANGE COLORS
------------------------------------------------------------

Open css/style.css and locate the :root section at the top:

  :root {
    --primary:      #FF6B35;   ← Main accent color (orange)
    --secondary:    #1a1a2e;   ← Dark navy color
    --text:         #2d2d3f;   ← Body text color
    ...
  }

Change the hex codes to your preferred colors.
All elements using these variables will update automatically.


------------------------------------------------------------
  HOW TO ADD PRODUCTS
------------------------------------------------------------

In shop.html, copy a product card block:

  <div class="col-6 col-md-4">
    <div class="product-card" data-product-id="YOUR_ID"
         data-product-name="Product Name"
         data-product-price="29.99">
      <div class="product-img-wrapper">
        <img src="images/your-image.png" alt="Product Name"/>
      </div>
      <div class="product-body">
        <div class="product-name">Product Name</div>
        <div class="price-current">$29.99</div>
        <button class="btn-add-cart">Add to Cart</button>
      </div>
    </div>
  </div>

Replace the values with your product details.


------------------------------------------------------------
  CART SYSTEM
------------------------------------------------------------

The shopping cart is powered by JavaScript (localStorage).
Cart data is saved in the browser — no backend required.

Features:
  ✓ Add to cart from any product card
  ✓ Cart count updates in real-time
  ✓ Quantity adjustment
  ✓ Remove items
  ✓ Coupon code support (SAVE10 = 10% off)
  ✓ Order total calculation


------------------------------------------------------------
  HOW TO UPLOAD YOUR WEBSITE
------------------------------------------------------------

1. Purchase web hosting (Hostinger, Namecheap, Bluehost, etc.)
2. Log in to your hosting control panel.
3. Open the File Manager.
4. Navigate to the "public_html" or "www" folder.
5. Upload all files from the "ecommerce-template" folder.
6. Your website will be live at your domain!

Recommended Hosting Providers:
  - Hostinger → https://hostinger.com
  - Namecheap → https://namecheap.com
  - Netlify    → https://netlify.com (Free for static sites)
