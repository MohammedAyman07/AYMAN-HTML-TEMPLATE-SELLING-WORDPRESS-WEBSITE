HOW TO EDIT:
--------------------------------------------------
1. Colors & Theme:
   Open `css/style.css` and modify the variables at the top under `:root`.
   Example:
   --primary: #FF6B2C;
   --accent: #0066FF;

2. Images:
   Replace the images in the `images` folder or update the `src` paths in the HTML files.
   For product images, standard ratios like 1:1 or 4:5 work best.

3. Cart System:
   The cart uses LocalStorage to save items via `js/script.js`.
   Products are added by clicking buttons with the `data-add-cart` attribute.
