ArtHouse - Premium Art Ecommerce Template
=========================================

Thank you for choosing the ArtHouse ecommerce template!

Description
-----------
ArtHouse is a modern, premium, and fully responsive HTML5 template designed specifically for artists, art galleries, digital creators, and handmade craft sellers. It features a clean aesthetic, smooth animations, and a focus on highlighting artwork beautifully.

Folder Structure
----------------
art-ecommerce-template/
│
├── index.html       - Home page with hero, categories, and featured art
├── shop.html        - Main product grid with filter layout
├── artwork.html     - Single product/artwork detail page
├── cart.html        - Shopping cart overview
├── checkout.html    - Checkout form and order summary
├── contact.html     - Contact information and form
│
├── css/
│   └── style.css    - Main stylesheet with custom properties
│
└── js/
    └── script.js    - Template interactions (cart, sticky nav, gallery)

Features
--------
* Built with Bootstrap 5
* Fully Responsive Design
* CSS Variables for easy color customization (see style.css :root)
* Custom CSS grid and Flexbox layouts
* Font Awesome 6 Icons
* Google Fonts (Cormorant Garamond & Montserrat)
* Smooth scroll animations
* Complete e-commerce page flow (Shop -> Details -> Cart -> Checkout)

Customization
-------------
Colors and Fonts can be easily updated in the `css/style.css` file under the `:root` variables section at the top of the document.

Images
------
The template currently uses high-quality CSS gradient placeholders and FontAwesome icons as images to demonstrate the layout. You can replace these by adding proper `<img>` tags or updating the background-images in CSS to point to your actual artwork images.
