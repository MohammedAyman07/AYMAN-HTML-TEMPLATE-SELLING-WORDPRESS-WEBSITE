LUXE FASHION - Premium HTML5 Ecommerce Template
==================================================

Thank you for downloading LUXE Fashion template!

FOLDER STRUCTURE
----------------
index.html       - Homepage (Hero, Categories, Featured Products, New Arrivals)
shop.html        - Product listing page with sidebar filters
product.html     - Product details page with image gallery and description
cart.html        - Shopping cart page with dynamic JS updates
checkout.html    - Premium checkout process page
contact.html     - Contact us page with interactive form
css/style.css    - Main typography, layouts, colors, and responsive design
js/script.js     - Cart functionality, animations, sliders, tab filters, validation
images/          - Project image assets folder

FEATURES
----------------
- Built with HTML5, CSS3, Vanilla JS
- Bootstrap 5 framework for layout
- Stylish, minimal, and modern design
- Fully responsive across all devices
- Working add-to-cart Javascript logic using LocalStorage
- Smooth scroll animations using IntersectionObserver
- Multi-step styled checkout page
- Clean and well-commented code
- Elegant typography with hover animations
- W3C validated code

CUSTOMIZATION
----------------
- Colors: Open css/style.css and modify the CSS variables in the :root section on line 11 onwards.
  Example: change --gold: #c9a84c; to your brand accent color.
- Fonts: The template uses Google Fonts (Cormorant Garamond and Inter). Change the @import URL in style.css to use your preferred fonts.
- Images: Replace the Unsplash placeholder image URLs in the HTML files with your own product image paths.
