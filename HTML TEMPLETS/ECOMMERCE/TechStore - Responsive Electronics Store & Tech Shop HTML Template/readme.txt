TechStore - Electronics Ecommerce Template
==========================================

Version: 1.0
Author: TechStore Template

Description
-----------
A modern, premium HTML5 ecommerce website template designed specifically for electronics stores, gadget shops, and technology sellers. The template is built using HTML5, CSS3, Bootstrap 5, and JavaScript.

Features
--------
- 6 Fully Responsive HTML Pages
- Modern Dark Mode Tech UI
- Custom CSS with Variables (No heavy frameworks besides Bootstrap grid/utils)
- Shopping Cart functionality (localStorage based)
- Wishlist functionality (localStorage based)
- SVG placeholder icons for product images
- Smooth Scroll & CSS Animations
- Countdown timer for Deals section

Folder Structure
----------------
electronics-ecommerce-template/
├── index.html        (Homepage)
├── shop.html         (Product Grid/Listing)
├── product.html      (Product Detail Page)
├── cart.html         (Shopping Cart)
├── checkout.html     (Checkout Form)
├── contact.html      (Contact Us)
├── css/
│   └── style.css     (Main Stylesheet)
├── js/
│   └── script.js     (Interactions, Cart Logic)
└── images/           (Folder for user's actual product images)

Setup Instructions
------------------
1. Download or extract the template folder.
2. Open `index.html` in any web browser to view the template locally.
3. No build tools or servers are required for viewing. For production deployment, simply upload the contents of the `electronics-ecommerce-template` folder to your web server.

Image Placeholders
------------------
This template uses Font Awesome SVG icons mixed with modern CSS gradients for product images to maintain a clean aesthetic before actual user product photography is added. To add your own images, replace the `<div class="product-img-wrap">...</div>` contents with standard `<img src="path/to/image.jpg" alt="Product">` tags.
