HOW TO EDIT THE WEBSITE

1. Open the template folder.
2. Open any HTML file (index.html, about.html, etc.).
3. Use a code editor like:
   - Visual Studio Code
   - Sublime Text
   - Notepad++

4. Edit text content inside the HTML tags.

Example:
<h1>Welcome to Our Restaurant</h1>

You can change it to:
<h1>Welcome to Your Restaurant Name</h1>


HOW TO CHANGE IMAGES

1. Go to the images folder.
2. Replace existing images with your own images.
3. Use the same image name to keep layout working.


HOW TO CHANGE COLORS

1. Open css/style.css
2. Find color codes like:

#d4a762

3. Replace with your preferred color.


HOW TO UPLOAD WEBSITE TO HOSTING

Step 1:
Buy hosting from any provider like:
Hostinger / Bluehost / Namecheap

Step 2:
Login to your hosting control panel.

Step 3:
Open File Manager.

Step 4:
Go to the public_html folder.

Step 5:
Upload all template files inside public_html.

Step 6:
Your website will be live.
