==============================
HOW TO CHANGE IMAGES
==============================

The template uses images from Unsplash.com via
URL links. To replace with your own images:

Method 1 (Recommended):
  - Place your images in the /images/ folder.
  - Replace the Unsplash URL in the img src=""
    attribute with your image path, e.g.:
    src="images/my-photo.jpg"

Method 2:
  - Continue using Unsplash URLs or swap
    with any other image hosting URL.

Image Tips:
  - Hero images: use at least 1400x900px
  - Blog card thumbnails: 800x500px recommended
  - Author avatar: square images (e.g., 200x200px)

==============================
HOW TO CHANGE THE ACCENT COLOR
==============================

Open css/style.css and change the following
CSS variable at the top of the file:

  :root {
    --primary: #ff6b35;       /* Main accent color */
    --primary-dark: #e55a24;  /* Darker accent      */
    --primary-light: #fff3ee; /* Light tint          */
  }

Change #ff6b35 to any hex color code you like.
For example:
  - Blue   : #2563eb
  - Green  : #16a34a
  - Purple : #7c3aed

==============================
HOW TO CHANGE FONTS
==============================

The template uses Google Fonts loaded via a
CSS @import at the top of style.css:

  @import url('https://fonts.googleapis.com/css2?
    family=Playfair+Display:...
    family=Inter:...');

To change fonts:
1. Go to https://fonts.google.com
2. Select your preferred fonts.
3. Copy the @import URL and replace the existing
   one in style.css.
4. Update the font-family values in the :root
   variables (--font-body and --font-heading).
