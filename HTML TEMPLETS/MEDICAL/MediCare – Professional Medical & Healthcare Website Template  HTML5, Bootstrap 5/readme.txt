HOW TO EDIT THE WEBSITE
----------------------------------------

1. Open the template folder.
2. Open any .html file using a code editor.

Recommended Editors:
- Visual Studio Code (recommended)
- Sublime Text
- Notepad++

Edit the text content inside the HTML tags.

----------------------------------------
HOW TO CHANGE IMAGES
----------------------------------------

1. Add your images to the /images/ folder.
2. In HTML files, find <img> tags and update
   the src attribute to your new image path.

Example:
<img src="images/your-image.jpg" alt="Description">

----------------------------------------
HOW TO CHANGE COLORS
----------------------------------------

1. Open css/style.css
2. Find the :root section at the top of the file
3. Modify the CSS variable values

Primary Color  : --primary
Accent Color   : --accent
Background     : --light
Text Color     : --text-main

Example:
--primary: #0d6efd;   Change to your brand color

HOW TO UPLOAD YOUR WEBSITE
----------------------------------------

5. Upload all template files
