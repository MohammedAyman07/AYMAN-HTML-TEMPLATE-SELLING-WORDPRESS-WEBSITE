AYMAN INFOTECH WordPress Theme
================================

Installation Instructions:

1. Upload the `ayman-infotech` folder to your WordPress `wp-content/themes/` directory.
2. Log in to your WordPress Dashboard.
3. Go to Appearance > Themes.
4. Activate the "AYMAN INFOTECH" theme.
5. Install and activate WooCommerce plugin.

Setup:

1. Create Pages:
   - Create a page named "Home" and set its template to "Front Page".
   - Create a page named "About Us" and set its template to "About Us".
   - Create a page named "Contact" and set its template to "Contact Us".
   - Go to Settings > Reading and set "Your homepage displays" to the "Home" page you just created.

2. WooCommerce Settings:
   - Go to Products > All Products and add your digital products.
   - For each product, use the "Live Demo settings" box in the sidebar to add a demo URL.
   - Go to Appearance > Menus and assign menus to "Primary Menu" and "Footer Menu".

3. Live Demo:
   - To make the Live Demo button appear, ensure you fill in the "Live Demo URL" field when editing a product.

Enjoy your premium digital marketplace!
